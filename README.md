# 🏥 АкваМед — Клиника капельниц в Санкт-Петербурге

> **Комплексная веб-система клиники инфузионной терапии:** современный Landing Page + интерактивная CRM-панель + Telegram-бот для мгновенного управления заявками.

Проект представляет собой готовое решение для автоматизации медицинских услуг на дому и в клинике. Он объединяет адаптивную клиентскую часть, защищенный серверный обработчик на PHP и Telegram-бота, заменяющего громоздкие CRM-системы для операторов.

---

## 🚀 Основные возможности

### 📱 Клиентская часть (Frontend)
- **Современный адаптивный дизайн:** Лендинг полностью оптимизирован под мобильные устройства (от 360px) и экраны Retina.
- **Интерактивные формы:**
  - **Drip Sheet (Шторка подбора капельниц):** выезжающее сбоку окно с формой заказа конкретных процедур на дом.
  - **Модальное окно консультации:** всплывающая форма для записи на прием.
  - **Баннерная форма:** встроенная форма быстрого заказа бесплатной консультации.
- **Умная маска ввода телефона:** автоподстановка `+7` и форматирование вида `+7 (999) 999-99-99` с защитой от ввода букв.
- **Глобальный поиск с подсказками:**
  - Быстрый поиск услуг и капельниц по ключевым словам и симптомам (например, «похмелье», «золушка», «красота»).
  - Теги-подсказки для мгновенного заполнения поиска.
  - Выпадающий список с живой фильтрацией результатов.
- **Мягкие слайдеры и аккордеоны:**
  - **Лицензии:** карусель документов с плавной прокруткой и отзывчивой навигацией.
  - **FAQ Accordion:** интерактивный раздел часто задаваемых вопросов с анимацией стрелок.
  - **Mobile Menu Accordion:** многоуровневое мобильное меню с аккордеонами для категорий услуг.

### ⚙️ Серверная логика (Backend & DB)
- **Надежная защита:** Вставка заявок в MySQL с использованием подготовленных выражений (**PDO prepared statements**), исключающих SQL-инъекции.
- **Анти-спам фильтр:** Ограничение на повторную отправку заявок с одинаковыми контактами чаще, чем раз в 5 минут.
- **Административные таблицы:**
  - `get_requests.php` — просмотр всех заявок с сортировкой по новизне.
  - `get_subscribers.php` — список зарегистрированных пользователей Telegram-бота.

### 🤖 Telegram CRM Бот (`@BotFather` API)
- **Подписка на лиды:** Любой администратор может активировать CRM-режим командой `/start` и мгновенно получать новые заявки.
- **Удобные карточки лидов:**
  - Вывод имени, email и телефона в моноширинном формате (`<code>`) для копирования в один клик на смартфоне.
  - Указание уникального ID заявки.
- **Интерактивное управление:**
  - Кнопка **«✅ Отметить обработанной»** — переводит заявку в статус `processed` в БД и динамически обновляет сообщение бота.
  - Кнопка **«💬 Оставить комментарий»** — запрашивает текст через `ForceReply` и крепит примечание к заявке в БД.
- **Быстрый доступ через меню и клавиатуру:**
  - `🟢 Активные заявки` — список новых заявок с быстрыми ссылками `/manage_ID`.
  - `📋 Все заявки` — история последних 15 заявок.
  - `✅ Обработанные` — архив завершенных лидов.
  - `💬 С комментариями` — просмотр заявок, содержащих примечания оператора.

## 🛠️ Стек технологий

| Компонент | Технологии |
| :--- | :--- |
| **Frontend** | HTML5, CSS3 (Vanilla / Custom Flexbox & Grid), JavaScript (ES6+) |

| **Контейнеры** | Docker, Docker Compose |

## ⚙️ Установка и запуск

Для локального запуска вам понадобятся **Docker** и **Docker Compose**.

### Шаг 1: Клонирование репозитория
```bash
git clone https://github.com/Gabbe1a/Kapelnica-med.git
cd Kapelnica-med
```

### Шаг 2: Настройка переменных окружения
Создайте файл `.env` в корневой папке проекта на основе примера:
```bash
cp .env.example .env
```
Откройте файл `.env` и укажите токен вашего Telegram-бота:
```env
TG_BOT_TOKEN=1234567890:ABCdefGhIJKlmNoPQRsTUVwxyZ
```

### Шаг 3: Запуск контейнеров
Запустите проект в фоновом режиме:
```bash
docker compose up -d
```
Docker автоматически:
1. Скачает образы PHP 8.2 Apache и MySQL 8.0.
2. Создаст базу данных `kapelnica_db` и применит к ней структуру из `backend/schema.sql`.
3. Подключит проект и запустит локальный веб-сервер на порту `:80`.
4. Запустит сервис `tg_bot`, который начнет опрашивать сервера Telegram.

## 🕹️ Использование системы

1. **Лендинг клиники:** доступен в браузере по адресу `http://localhost`.
2. **Административная часть (заявки):** `http://localhost/backend/get_requests.php`.
3. **Начало работы с Telegram-ботом:**
   - Найдите вашего бота в Telegram и отправьте команду `/start`.
   - Бот зарегистрирует ваш `chat_id` и выдаст удобную клавиатуру меню.
   - Оставьте тестовую заявку на сайте — бот мгновенно пришлет карточку лида со звуковым уведомлением.

## 🔒 Безопасность и оптимизация
- Все формы используют защиту от XSS (через очистку строк `trim` и вывод с помощью `htmlspecialchars`).
- Подключение к БД защищено от утечки учетных записей (все настройки вынесены в `db.php` для внутренних Docker-линков).
- Доступна очистка кэша статики на лету с помощью версионирования CSS/JS (`?v=8`).
